# Oriora Apps

Downloads for the on-device apps built by **[Oriora](https://orioralabs.com)**.

Each app is native and **local-first**: your data stays on your own device, and AI
runs through your Oriora account and wallet. One Oriora login works across every app.

## How downloads work

Every app is published as a versioned **[Release](../../releases)** on this repo. Find the
app's latest release, download the installer for your platform, and open it.

- App **source code stays in private per-app repositories** — only the compiled installer is published here.
- macOS installers are signed with an Apple Developer ID and notarized by Apple before release.
